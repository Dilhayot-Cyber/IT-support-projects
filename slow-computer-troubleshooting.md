## Extended Description
This project demonstrates how IT Support and Help Desk technicians diagnose and fix slow computer performance issues.
Slow computers are one of the most common problems in office environments. This guide explains step-by-step methods used by IT professionals to identify the cause and improve system performance.
Topics covered in this project include:
- Startup programs
- High CPU usage
- Low RAM
- Disk space issues
- Malware infections

---

## Problem: Computer Running Very Slow

### Possible Causes

- Too many startup programs
- High CPU usage
- Low RAM
- Full hard drive
- Malware

### Troubleshooting Steps

1. Restart the computer
2. Open Task Manager
3. Check CPU and memory usage
4. Disable unnecessary startup programs
5. Run Disk Cleanup
6. Scan for malware
