## Extended Description

This project demonstrates common printer troubleshooting techniques used by IT Support and Help Desk technicians.

Printer issues are one of the most frequent problems in office environments. This guide explains how to diagnose and resolve common printer problems step by step.

Topics covered include:

- Printer not printing
- Printer offline issue
- Print queue stuck
- Driver problems
- ---

## Problem: Printer Not Printing

### Possible Causes

- Printer is turned off
- USB or network cable disconnected
- Printer driver issue
- Paper jam

### Troubleshooting Steps

1. Check if the printer is powered on
2. Check USB or network connection
3. Restart the printer
4. Restart the computer
5. Check for paper jams
6. Reinstall printer drivers
---

## Problem: Printer Offline

### Troubleshooting Steps

1. Open Control Panel
2. Go to Devices and Printers
3. Right click the printer
4. Select "Set as default printer"
5. Disable "Use printer offline"
